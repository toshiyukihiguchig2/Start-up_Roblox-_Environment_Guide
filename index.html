GitHubの準備

2.1. アカウント作成とGitインストール

  1. 1
    GitHubアカウント作成 GitHub.com でアカウントを作成してください。
  2. 2
    Gitインストール Git for Windows をダウンロード・インストール。
  3. 3
    初期設定 PowerShellを開き、以下を実行してください。
powershell 
git config --global user.name "YourName"
git config --global user.email "your-email@example.com"

2.2. リポジトリの作成

  1. 1
    [New repository] を作成GitHubのダッシュボードから新規リポジトリを作成。
  2. 2
    Repository nameroblox-project-name（任意の名前）
  3. 3
    Public / Privateプロジェクトに合わせて選択。
  4. 4
    Initialize「Add a README file」にチェック、「.gitignore」は Roblox を選択。
  5. 5
    URL をコピー緑色の [<> Code] ボタンからHTTPS URLをコピー。

4.1. サーバー起動と同期

  1. 1
    Rojo サーバーを起動 VS Codeのターミナルで以下を実行：
powershell 
rojo serve
  1. 2
    Studio と接続 Roblox Studioを開き、Rojoプラグインから「Connect」を押下。
✅
src/ 内のコードが Studio の ReplicatedStorage 等へリアルタイム同期されます。

4.2. テストの実行 (TestEZ)

  1. 1
    依存関係の導入wally install を実行し Packages/ を生成。
  2. 2
    テストコード作成src/tests/ 内に .spec.lua ファイルを作成。
  3. 3
    実行Studioの「プレイ (F5)」を押すと、TestRunner スクリプトにより自動でテスト結果が出力されます。

CI (GitHub Actions) の設定

リポジトリの .github/workflows/test.yml に以下を配置することで、pushのたびに自動テストが走ります。

yaml 
name: Test
on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Install Foreman (Tools)
        run: |
          curl -L https://github.com/Roblox/foreman/releases/latest/download/foreman-linux-x86_64.zip -o foreman.zip
          unzip foreman.zip
          chmod +x foreman
          sudo mv foreman /usr/local/bin/
      - name: Install Wally & Rojo
        run: foreman install
      - name: Add Path
        run: echo "$HOME/.foreman/bin" >> $GITHUB_PATH
      - name: Build & Verify
        run: rojo build test.project.json -o test.rbxl

.gitignore の徹底

⚠️
以下のファイルは 絶対にGitに含めない でください（リポジトリの肥大化や競合を防ぐため）。
対象理由
.rbxl / .rbxmRobloxバイナリファイル — サイズが大きく差分管理が困難
Packages/Wallyで管理するため、コミット不要
.vscode/個人設定 — チーム間で競合が発生する
